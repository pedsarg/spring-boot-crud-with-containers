# Project Status

**Under Development**

## Overview

This project aims to implement a simple CRUD (Create, Read, Update, and Delete) application using a containerized architecture with Docker.

The frontend is being developed with **Angular**, while **MySQL** is used as the database.

## 🛠️ Technologies & Tools

*   **Back-end:** Java 25, Spring Boot 4.1.0 (Spring Data JPA, Spring Web)
*   **Database:** MySQL
*   **DevOps:** Docker & Docker Compose
*   **Front-end:** Angular / HTML5 / CSS *(Developed with the assistance of AI/LLMs for agile interface prototyping)*

---

### 🤖 Note on Front-end Development
Since this project primarily focuses on the **Back-end (Spring Boot) and DevOps (Docker)** ecosystem, I utilized Artificial Intelligence tools to accelerate the creation and design of the user interface (Front-end). 

This approach allowed me to focus heavily on the API architecture, database modeling, and systems integration, leveraging AI as a productivity tool to deliver a complete, fully-functional Full-stack application.

## Features

The application includes:

* User authentication through a **Login Page**;
* A **Registration Page** for creating new records;
* A **Management Page** that allows users to:

  * Search records;
  * Edit existing data;
  * Delete records.

The main goal of this project is to provide a complete example of a full-stack CRUD application integrated with Docker, Angular, and MySQL.
